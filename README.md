# Lumen

Lecteur musical avec interface Electron (desktop) et version web PWA (iPhone / navigateur).

## Application desktop (Linux)

```bash
npm install
npm start
```

## PWA web (iPhone)

La version web est dans le dossier `docs/`. Elle est publiée automatiquement sur GitHub Pages à chaque push sur `main`.

URL : **https://leanuhq.github.io/lumen/**

Sur iPhone : ouvrir l’URL dans Safari → Partager → **Sur l’écran d’accueil**.

## Développement

- `npm start` — lance l’app Electron
- `npm run dev` — Electron avec URL de dev (port 3000)
- `npm run dist` — build installateur Linux
